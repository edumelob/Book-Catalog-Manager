#  Gerenciador de Catálogo de Livros

Um projeto simples em **Python** para praticar o uso de `namedtuple`, manipulação de arquivos e lógica básica de CRUD (Create, Read, Update, Delete).

---

##  Tecnologias usadas
- Python 3+
- `collections.namedtuple`

---

##  Funcionalidades
- Adicionar livros ao catálogo
- Listar todos os livros
- Buscar livros pelo título
- Persistência dos dados em arquivo `.txt`
